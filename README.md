# The Rational Budget
💡 This webapp will help you manage your budget and save your precious money 💡<br>
🖥️ Access it through this link 👉 https://the-rational-budget.onrender.com/<br><br>
![the rational budget](https://github.com/user-attachments/assets/fa46a7ea-b6b4-44f4-b553-598a539b1c62)<br>
## Features
🌐 Tech stack: MERN (MongoDB, Express.js, React.js, Node.js) + Apollo GraphQL<br>
🔧 Learn type definitions and resolvers to define the GraphQL schema and data fetching logic<br>
✍️ Implement mutations for modifying data and establishing relationships in the graph<br>
🔒 Secure authentication using Passport.js with a MongoDB session store<br>
📦 Manage global state with Apollo Client<br>
⚠️ Handle errors on both the server and client sides<br>
🚀 Simplify deployment with Render platform<br>
⏱️ Automate tasks with cron jobs<br>
### Setup .env file
<pre>MONGO_URI=...
SESSION_SECRET=...</pre>
### Build the app
<pre>npm run build</pre>
### Start the app
<pre>npm start</pre>
